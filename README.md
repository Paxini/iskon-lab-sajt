# Iskon Lab - Website

Moderne, animirani sajt za Iskon Lab - programersku i marketinšku agenciju.

## Tehnologije

- **React 18** - UI framework
- **Vite** - Build tool
- **Framer Motion** - Animacije
- **Tailwind CSS** - Styling
- **Lenis** - Smooth scrolling

## Instalacija

```bash
# Instaliraj dependencies
npm install

# Pokreni development server
npm run dev

# Build za produkciju
npm run build

# Preview production builda
npm run preview
```

## Struktura projekta

```
src/
├── components/
│   ├── Hero/           # Hero sekcija sa animiranim logom
│   ├── Services/       # Usluge sa 3D karticama
│   ├── Portfolio/      # Portfolio sa filter animacijama
│   ├── Contact/        # Kontakt forma
│   └── ui/             # Reusable UI komponente
├── hooks/              # Custom React hooks
├── App.jsx             # Glavna komponenta
├── main.jsx            # Entry point
└── index.css           # Globalni stilovi
```

## Karakteristike

- **Custom Cursor** - Animirani cursor sa kontekstualnim tekstom
- **Smooth Scroll** - Lenis-powered buttery smooth scrolling
- **3D Cards** - Kartice sa tilt efektom na hover
- **Magnetic Buttons** - Dugmad koja prate kursor
- **Scroll Animations** - Elementi se animiraju pri ulasku u viewport
- **Layout Animations** - Smooth tranzicije pri filtriranju portfolia
- **Responsive Design** - Optimizovano za sve uređaje
